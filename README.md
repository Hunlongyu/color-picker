<h1 align="center">Color Picker 取色器</h1>

### ✨特性
1. 一键取色并复制到剪贴板
2. 保留历史取色记录
3. 自定义颜色值类型（如：HEX, RGB 等）
4. 自定义取色快捷键（默认：F2）
5. 颜色值去重
6. 历史记录：左键单击显示颜色值，左键双击复制颜色值到剪贴板，右键单击删除颜色
7. 可隐藏界面取色
8. 软件安装包小，占用内存少

1. 软件默认只支持 WIN11。低于 WIN11 会在安装过程中安装运行环境。
2. 下拉菜单会错位显示，这个是底层框架的 BUG，目前还没有解决方案。
